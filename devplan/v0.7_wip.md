### M656 ‚Äî `_call_role()` wrapper for simple LLM roles

**Problem:** `run_summarizer`, `run_paraphraser`, `run_searcher`, `run_exec_translator`
all repeat the same 6-line pattern:
```python
messages = build_XXX_messages(...)
try:
    return await call_llm(config, "role", messages, session=session)
except LLMError as e:
    raise XxxError(f"...: {e}")
```

4 functions √ó 6 lines = 24 lines of identical boilerplate.

**Files:** `kiso/brain.py`

**Change:**

- [ ] Extract helper:
      ```python
      async def _call_role(config, role, messages, error_class, session=""):
          try:
              return await call_llm(config, role, messages, session=session)
          except LLMError as e:
              raise error_class(f"LLM call failed: {e}")
      ```
- [ ] Rewrite `run_summarizer`, `run_paraphraser`, `run_searcher`,
      `run_exec_translator` as 2-line wrappers
- [ ] Existing tests pass unchanged

**Savings:** ~40 LOC

### M660 ‚Äî Extract `_execute_with_review()` from task handlers

**Problem:** `_handle_exec_task`, `_handle_tool_task`, `_handle_search_task` all share
the same retry+review loop structure (~50 lines each):
```
while True:
    execute task (type-specific)
    run reviewer
    if stuck ‚Üí return stuck result
    if replan + retry_hint + retries left ‚Üí rebuild context, continue
    if replan ‚Üí return replan result
    if ok ‚Üí break
finalize (audit, update, webhook)
```

The loop logic is identical ‚Äî only the execution step differs.

**Files:** `kiso/worker/loop.py`

**Change:**

- [ ] Extract `_execute_with_review(ctx, task_row, execute_fn, rebuild_fn)` that
      encapsulates the retry/review loop
- [ ] `execute_fn(ctx, task_row) ‚Üí (stdout, stderr, exit_code)` ‚Äî type-specific
- [ ] `rebuild_fn(ctx, task_row, retry_hint) ‚Üí updated_detail` ‚Äî type-specific
- [ ] Each handler becomes: pre-flight checks + call `_execute_with_review()`
- [ ] Existing worker tests pass unchanged
- [ ] **Risk: medium** ‚Äî handlers have subtle differences in error paths, audit
      fields, and output formatting. Must preserve all edge cases.

**Savings:** ~100-150 LOC
