screenshot + publish + URL reachability. When it fails, impossible to tell which part broke.

**Files:** `tests/functional/test_browser.py`

**Change:**

- [ ] Split into 3 focused tests:

      - `test_browser_navigate_and_describe` — navigate to a URL, get text description

- [ ] Each test is independent (no session-scoped fixture coupling)

- [ ] Keep install test as a separate, slower test
