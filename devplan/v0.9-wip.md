## Phase 1 — Code deduplication (M1236-M1238)

### M1236 — Unify fallback switching in `_run_text_role` ✅

**Problem:** The `LLMStallError` and `LLMError` (timeout) handlers in
`_run_text_role` have near-identical 6-line fallback-switching blocks.
If one changes, the other gets forgotten.

**Fix:** Extract the common fallback-switch logic into a single block
with a boolean condition. Keep the divergent behavior (stall breaks if
no fallback, timeout retries same model) but eliminate the copy-paste.

**Tasks:**
- [x] Unified into single `except LLMError` with `should_fallback` boolean
- [x] 3983 unit tests pass

### M1238 — Extract text stats regex to conftest constants ✅

**Problem:** The multilingual chars/lines regex patterns are duplicated
in `test_multitool.py` and `test_preset_workflows.py`. If a new
language needs support, both files need independent updates — exactly
the bug we hit twice (M1223 fixed one file, M1228 caught the other).

**Fix:** Define `CHARS_COUNT_RE` and `LINES_COUNT_RE` as compiled
patterns in `tests/functional/conftest.py` and import in both test
files.

**Tasks:**
- [x] Added `CHARS_COUNT_RE` and `LINES_COUNT_RE` to `tests/functional/conftest.py`
- [x] Replaced inline patterns in `test_multitool.py` and `test_preset_workflows.py`
- [x] Tests collect cleanly
